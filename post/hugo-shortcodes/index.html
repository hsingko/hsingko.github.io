Hugo Shortcode简介

Written by hsingko with ♥ on November 8, 2020 in notes, 739 words

shortcodes其实就是hugo提供的一种增强markdown表现力的工具。

hugo自带的shortcode

其实shortcode具体是什么东西没必要搞清楚，知道它能干什么就好，它可以：

1.插入视频

目前支持vimeo，youtube。以youtube为例，比如你想要分享链接

https://www.youtube.com/watch?v=CDtQ3rai9Ks

那么只要在markdown中输入：

{{< youtube CDtQ3rai9Ks >}}

2.插入图片

这个功能不怎么有用，还不如markdown的简洁。唯一的优点是可以自定义图片的长宽。

{{< figure src="/images/cyberpunk/cyberpunk.jpg" title="cyberpunk" height=800 width=600 >}}

3.插入gist

目标：

https://gist.github.com/spf13/7896402

shortcode:

{{< gist spf13 7896402 >}}

你也可以自己编写(抄袭)shortcode

作为我这种资深伸手党兼菜鸡码农，代码自然是能抄就抄，很多功能丰富的模板都自带了一些好用的shortcode，我们只要知道从那里copy就可以了。目录是在：

themes/<victim-theme>/layouts/shortcodes/*

文件名自然就是shortcode的代码了，下面介绍一些比较有用的功能。

示例：插入html

在shortcodes目录下新增如下文件：

<!--shortcodes/rawhtml.html-->
<!-- raw html -->
{{.Inner}}

这样我们就拥有了一个名为{{rawhtml}}的shortcode，可以在这个标签内直接写入html代码。

代码：

{{< rawhtml >}}
<div>
   <style type="text/css" scoped>
       button { color: green; } 
   </style>
   <script type="text/javascript">
      var clicks = 0;
      function onClick() {
          clicks += 1;
          document.getElementById("clicks").innerHTML = clicks;
      };
   </script>
   <button type="button" onClick="onClick()">Click me</button>
   <p>Clicks: <a id="clicks">0</a></p>
</div>
{{< /rawhtml >}}

效果：

Clicks: 0

这个功能在想写一些自定义页面的时候比较好用，比如这个页面就是用html写的，是一个抑郁症自测表，有兴趣的同学可以做一下看自己能得几分。
